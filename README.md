# Modelo-Conceitual-BD-Oficina
Modelo Conceitual de Banco de Dados – Oficina Mecânica

Este repositório apresenta o modelo conceitual de banco de dados desenvolvido para um sistema de controle e gerenciamento de ordens de serviço em uma oficina mecânica.

A modelagem foi construída com base na narrativa proposta, representando todos os elementos essenciais: clientes, veículos, equipes, mecânicos, serviços, peças e o funcionamento das ordens de serviço.

✔ Decisão de Design: Modelo Conceitual em Formato de Tabelas

Além do diagrama visual clássico, optei por criar também uma versão do modelo conceitual em formato de tabelas.
Esse formato foi escolhido porque algumas entidades possuem diversos atributos e, dessa maneira, a visualização se torna:

mais harmoniosa,

mais clean,

mais organizada,

mais prática de consultar,

e mais clara quanto aos relacionamentos.

O modelo conceitual tabular mantém o foco no entendimento, sem inclusão de PK ou FK (por se tratar de um nível conceitual).

✔ Papel Central da Ordem de Serviço

A Ordem de Serviço é o elemento central do sistema e organiza todo o fluxo de funcionamento da oficina.
A partir dela é possível:

associar os veículos e seus respectivos clientes,

registrar os serviços executados,

vincular peças utilizadas,

acompanhar o trabalho da equipe de mecânicos,

controlar valores e etapas,

estruturar a conclusão das atividades.

Essa abordagem centralizada trouxe maior clareza ao modelo e refletiu melhor a dinâmica real do funcionamento da oficina.
